🃏Генератор флеш-карточек

Пример для 9 класса

Сделай 5 учебных флеш-карточек по теме "Причастие в русском языке" для 9 класса. С одной стороны — вопрос, с другой — ответ. Формат: для использования в Wordwall или вручную.

Пример для 7 класса

Сделай 5 учебных флеш-карточек по теме "Наречие" для 7 класса. С одной стороны — вопрос, с другой — ответ. Формат: для использования в Wordwall или вручную.
Применение: Используйте для разминки, повторения, индивидуальной работы с учениками

❓Провокационные вопросы

Вовлекайте учеников с первых минут урока играми "Истина/Ложь"

Базовый промпт:

Придумай 3 утверждения, одно из которых ложное, по теме "[ТЕМА]". Формат — для игры "Истина/Ложь".

Пример: Пунктуация

Придумай 3 утверждения, одно из которых ложное, по теме "Пунктуация при причастном обороте". Формат — для игры "Истина/Ложь".

Пример: Морфология

Придумай 3 утверждения, одно из которых ложное, по теме "Правописание НЕ с глаголами". Формат — для игры "Истина/Ложь".
Эффект: Активирует знания, вызывает интерес, создаёт интригу с первых минут урока
